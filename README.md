## 📚 Projeto Java – Consumo de API e Persistência de Dados

Este projeto foi desenvolvido utilizando **Java**, aplicando conceitos de **POJO**, **JDBC** e **PostgreSQL**, além do consumo de dados da **BrasilAPI** para obtenção de informações externas.

A aplicação realiza a integração entre a API e o banco de dados, permitindo consultar dados da BrasilAPI, tratá-los na aplicação e armazená-los no banco PostgreSQL utilizando conexão JDBC. O projeto também utiliza classes **POJO (Plain Old Java Objects)** para modelagem das entidades e organização da estrutura do código.

### 🚀 Tecnologias utilizadas

* Java
* POJO (modelagem de classes)
* JDBC (conexão com banco de dados)
* PostgreSQL
* Consumo de API REST – BrasilAPI
